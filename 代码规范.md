# C++ 代码规范

> 本规范参考了 Google C++ Style Guide、C++ Core Guidelines 以及业界最佳实践，专为初学者设计，包含清晰的示例和正反对比。

---

## 一、命名规范

### 1.1 基本原则

| 类型 | 命名风格 | 示例 |
|------|---------|------|
| 宏常量 | 全大写 + 下划线分隔 | `MAX_SIZE`, `PI` |
| 常量（`const`） | 全大写 + 下划线分隔 | `const int MAX_COUNT = 100;` |
| 变量名 | 小驼峰命名 | `studentName`, `totalScore` |
| 函数名 | 小驼峰命名 | `calculateArea()`, `printResult()` |
| 类名/结构体名 | 大驼峰命名 | `Student`, `Point2D` |
| 枚举名 | 大驼峰命名 | `enum Color { Red, Green, Blue };` |
| 枚举值 | 全大写 + 下划线分隔 | `Red`, `GREEN`, `BLUE`（见下文说明） |
| 文件命名 | 小写 + 下划线分隔 | `student_info.cpp`, `point_2d.h` |

### 1.2 正例与反例

```cpp
// ✅ 推荐
const int MAX_STUDENTS = 50;
int studentAge = 18;
void printStudentInfo(const string& name);
struct Point2D {
    int x;
    int y;
};

// ❌ 不推荐
const int max_students = 50;    // const常量应全大写
int StudentAge = 18;            // 变量名不应大写开头
void PrintStudentInfo();         // 函数名不应大写开头
struct point_2d;                // 结构体名应大驼峰
```

## 二、文件组织

### 2.1 文件结构

```
project/
├── src/
│   ├── main.cpp
│   ├── student.cpp
│   └── point.cpp
└── include/
    ├── student.h
    └── point.h
```

- 头文件（`.h`）：存放类/函数声明、常量定义
- 源文件（`.cpp`）：存放类/函数实现

### 2.2 头文件保护

**必须**使用 include guard 防止重复包含：

```cpp
// ✅ 推荐 - 使用 #pragma once（现代编译器支持）
#pragma once

// ✅ 推荐 - 使用传统 include guard
#ifndef STUDENT_H
#define STUDENT_H

// 头文件内容

#endif // STUDENT_H
```

### 2.3 头文件引用顺序

```cpp
// ✅ 推荐顺序（从内到外）
#include "student.h"              // 1. 当前项目头文件
#include <iostream>               // 2. C++标准库头文件
#include <cstdio>                 // 3. C标准库头文件
```

## 三、注释规范

### 3.1 注释原则

- **说明为什么，而不是做什么**：代码本身已经说明了做什么，注释应解释设计意图
- **保持注释与代码同步**：代码修改时，注释也要同步更新
- **简洁明了**：避免冗长的注释

### 3.2 注释类型

#### 3.2.1 单行注释

```cpp
// ✅ 推荐 - 说明设计意图或关键逻辑
int maxScore = 100;  // 期末考试满分

// ❌ 不推荐 - 废话注释
int maxScore = 100;  // 定义最大分数变量
```

#### 3.2.2 多行注释

```cpp
/*
 * ✅ 推荐 - 用于复杂逻辑的详细说明
 * 计算圆的面积，公式：S = π * r²
 * 参数r必须大于0，否则返回0
 */
double calculateCircleArea(double r) {
    if (r <= 0) return 0;
    return 3.1415926 * r * r;
}
```

#### 3.2.3 Doxygen 风格注释（推荐）

```cpp
/**
 * @brief 计算两个整数的和
 * 
 * @param a 第一个整数
 * @param b 第二个整数
 * @return int 两数之和
 */
int add(int a, int b) {
    return a + b;
}
```

## 四、变量与常量

### 4.1 变量声明

#### 4.1.1 声明位置

```cpp
// ✅ 推荐 - 在首次使用附近声明，提高可读性
void process() {
    int count = 0;
    for (int i = 0; i < 10; ++i) {
        count += i;
    }
}

// ❌ 不推荐 - 集中声明在函数开头（C语言风格）
void process() {
    int count;     // 距离使用位置太远
    int i;
    count = 0;
    for (i = 0; i < 10; ++i) {
        count += i;
    }
}
```

#### 4.1.2 变量初始化

**必须**在声明时初始化变量，避免使用未初始化的值：

```cpp
// ✅ 推荐 - 声明时初始化
int age = 0;
double score = 85.5;
char grade = 'A';
string name = "Unknown";

// ❌ 不推荐 - 未初始化
int age;              // 未定义行为！
double score;         // 未定义行为！
```

### 4.2 常量定义

```cpp
// ✅ 推荐 - 使用 const 定义常量（有类型检查）
const int MAX_SIZE = 100;
const double PI = 3.1415926;

// ⚠️ 可以使用但不推荐 - #define 宏（无类型检查）
#define MAX_SIZE 100
#define PI 3.1415926

// ✅ 推荐 - 枚举类型
enum class Color {
    Red = 1,
    Green = 2,
    Blue = 3
};
```

## 五、格式化风格

### 5.1 缩进

- 使用 **4个空格** 进行缩进（不要使用 Tab）
- 花括号 `{}` 独占一行

```cpp
// ✅ 推荐
if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

// ❌ 不推荐
if (score >= 60) { cout << "及格" << endl; }  // 花括号不独占一行
```

### 5.2 空格

- 运算符两侧各有一个空格
- 逗号后有一个空格
- 函数参数列表中，逗号后有一个空格

```cpp
// ✅ 推荐
int result = a + b * c;
void print(int a, int b);
for (int i = 0; i < 10; ++i) {}

// ❌ 不推荐
int result=a+b*c;           // 运算符两侧无空格
void print(int a,int b);    // 逗号后无空格
for (int i=0;i<10;++i) {}  // 运算符两侧无空格
```

### 5.3 空行

- 函数之间空一行
- 逻辑块之间空一行
- 不要在函数内过度使用空行

```cpp
// ✅ 推荐
int add(int a, int b) {
    return a + b;
}

int multiply(int a, int b) {
    return a * b;
}

void process() {
    int x = 10;
    int y = 20;
    
    int sum = add(x, y);
    int product = multiply(x, y);
    
    cout << sum << " " << product << endl;
}
```

## 六、函数规范

### 6.1 函数长度

- 单个函数长度控制在 **50-100行** 以内
- 超过100行的函数应拆分为多个小函数

### 6.2 函数参数

- 参数个数控制在 **4个以内**
- 复杂参数使用结构体或类封装

```cpp
// ✅ 推荐 - 参数简洁
double calculateArea(double radius);

// ✅ 推荐 - 使用结构体封装复杂参数
struct Rectangle {
    double width;
    double height;
};
double calculateArea(const Rectangle& rect);

// ❌ 不推荐 - 参数过多
double calculateSomething(int a, int b, int c, int d, int e);
```

### 6.3 返回值

- 返回值含义清晰，避免返回魔法数字
- 使用 `const` 引用返回大型对象，避免拷贝

```cpp
// ✅ 推荐 - 返回值含义清晰
bool isValid(int value) {
    return value >= 0 && value <= 100;
}

// ❌ 不推荐 - 魔法数字，难以理解
int checkValue(int value) {
    return 0;  // 0代表什么？
}
```

## 七、头文件规范

### 7.1 最小包含原则

头文件只包含必要的其他头文件，避免不必要的依赖：

```cpp
// ✅ 推荐 - 只包含必要的头文件
#include <string>
#include <vector>

// ❌ 不推荐 - 包含不必要的头文件
#include <iostream>  // 头文件中未使用
#include <string>
#include <vector>
#include <algorithm>  // 头文件中未使用
```

### 7.2 前向声明

使用前向声明避免包含头文件，减少编译时间：

```cpp
// ✅ 推荐 - 使用前向声明
class Student;  // 前向声明
void process(Student* student);

// ❌ 不推荐 - 直接包含头文件
#include "student.h"
void process(Student* student);
```

## 八、初学者常见问题

### 8.1 `using namespace std;`

```cpp
// ⚠️ 当前阶段可接受（简化代码）
#include <iostream>
using namespace std;

int main() {
    cout << "Hello" << endl;
    return 0;
}

// ✅ 推荐的长期做法（避免命名冲突）
#include <iostream>

int main() {
    std::cout << "Hello" << std::endl;
    return 0;
}
```

### 8.2 `system("pause")`

```cpp
// ⚠️ 当前阶段可接受（Windows环境）
system("pause");

// ✅ 推荐的跨平台做法
#include <iostream>

int main() {
    std::cout << "Press Enter to continue..." << std::endl;
    std::cin.get();
    return 0;
}
```

### 8.3 类型转换

```cpp
// ✅ 推荐 - 使用 C++ 风格转换
static_cast<double>(5) / 2;
const_cast<char*>(str.c_str());

// ⚠️ 当前阶段可接受 - C风格转换
(double)5 / 2;
```

## 九、规范示例：完整函数

### 9.1 不规范版本

```cpp
#include <iostream>
using namespace std;
#define MAX 100
int sum(int a[],int n){
    int s=0;
    for(int i=0;i<n;i++)s+=a[i];
    return s;
}
int main(){
    int arr[MAX];
    for(int i=0;i<MAX;i++)arr[i]=i;
    cout<<sum(arr,MAX)<<endl;
    system("pause");
    return 0;
}
```

### 9.2 规范版本

```cpp
#include <iostream>
#include <vector>

// 计算整数数组的和
// @param numbers 整数向量
// @return int 数组元素之和
int calculateSum(const std::vector<int>& numbers) {
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return sum;
}

int main() {
    const int MAX_SIZE = 100;
    std::vector<int> arr(MAX_SIZE);
    
    // 初始化数组：0, 1, 2, ..., 99
    for (int i = 0; i < MAX_SIZE; ++i) {
        arr[i] = i;
    }
    
    int result = calculateSum(arr);
    std::cout << "数组元素之和: " << result << std::endl;
    
    return 0;
}
```

## 十、初学者优先关注的3条

1. **变量声明时初始化**：永远不要使用未初始化的变量
2. **命名清晰**：变量名和函数名要能表达其用途，避免使用 `a`, `b`, `c` 等无意义的名称
3. **注释说明意图**：注释应解释"为什么这么做"，而不是"做了什么"
