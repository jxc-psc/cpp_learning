## 第一部分：数据类型与运算符

### 1.1 整数类型与补码截断（⭐）

**题目**：分析以下代码的输出结果，并解释原因。

```cpp
signed short num1 = 32768;
unsigned short num2 = -1;
signed char num3 = 257;
unsigned char num4 = 257;
std::cout << num1 << " " << num2 << " " << (int)num3 << " " << (int)num4 << std::endl;
```

**知识点**：补码截断机制、有符号/无符号转换公式

---

### 1.2 整数除法与隐式转换（⭐）

**题目**：编写程序，输入两个整数，输出它们的商（保留小数）和余数。

**要求**：
- 使用多种方法实现整数除法转浮点数除法
- 处理除数为零的情况

**知识点**：整数除法截断、隐式类型转换、`static_cast`

---

### 1.3 递增递减运算符（⭐）

**题目**：分析以下代码的输出结果。

```cpp
int x = 5;
int y = 5;
std::cout << ++x << " " << x++ << std::endl;
std::cout << --y << " " << y-- << std::endl;
```

**知识点**：前置递增/递减、后置递增/递减

---

### 1.4 位运算符应用（⭐⭐）

**题目**：使用位运算符实现以下功能：

1. 判断一个数是否为奇数
2. 将一个数的某一位置1
3. 将一个数的某一位清零
4. 交换两个整数（不使用临时变量）
5. 判断一个数是否为2的幂

**知识点**：位与(&)、位或(|)、位异或(^)、左移(<<)、右移(>>)

---

### 1.5 运算符优先级与结合性（⭐⭐）

**题目**：分析以下表达式的计算顺序和结果。

```cpp
int a = 1, b = 2, c = 3;
int result1 = a + b * c;
int result2 = (a + b) * c;
int result3 = a = b = c = 10;
int result4 = a > b ? a : b > c ? b : c;
```

**知识点**：运算符优先级、结合性、三目运算符右结合性

## 答案提示

> 以下是部分题目的答案提示，建议先尝试自己解决，再查看提示。

### 1.1 答案
```cpp
// num1: 32768 截断为 short 后，二进制最高位为1（负数），值为 -32768
// num2: -1 的补码为全1，无符号解析为 65535
// num3: 257 % 256 = 1，有符号解析为 1
// num4: 257 % 256 = 1，无符号解析为 1
// 输出：-32768 65535 1 1
```

### 1.4 答案提示
```cpp
// 判断奇数
bool isOdd(int n) { return (n & 1) == 1; }

// 置1第n位（从0开始）
int setBit(int n, int pos) { return n | (1 << pos); }

// 清零第n位
int clearBit(int n, int pos) { return n & ~(1 << pos); }

// 交换两个数
void swap(int& a, int& b) { a ^= b; b ^= a; a ^= b; }

// 判断2的幂
bool isPowerOfTwo(int n) { return (n & (n - 1)) == 0; }
```

### 4.4 金字塔答案提示
```cpp
int height = 5;
for (int row = 1; row <= height; ++row) {
    // 空格
    for (int space = 1; space <= height - row; ++space) {
        std::cout << " ";
    }
    // 星号
    for (int star = 1; star <= 2 * row - 1; ++star) {
        std::cout << "*";
    }
    std::cout << std::endl;
}
```

### 6.2 素数判断答案提示
```cpp
bool isPrime(int n) {
    if (n <= 1) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}
```

## 练习建议

1. **循序渐进**：从基础题开始，逐步挑战中等和进阶题
2. **动手实践**：每个题目都要编写代码验证，不要只看不动手
3. **注重细节**：注意数据类型选择、边界条件处理、错误处理
4. **代码规范**：按照之前学习的代码规范编写代码
5. **调试能力**：学会使用调试工具或添加输出语句调试程序
6. **优化思维**：完成基本功能后，思考如何优化算法和代码结构
