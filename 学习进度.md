### 核心知识点回顾

#### 1. C++程序基础结构
```cpp
#include <iostream>  // 包含输入输出流库
using namespace std; // 使用std命名空间

int main() {          // 主函数入口
    cout << "Hello, world" << endl;
    return 0;         // 返回0表示正常退出
}
```
- **关键理解**：`main()` 是程序执行的起点；`cout` 用于标准输出；`endl` 输出换行并刷新缓冲区

#### 2. 常量定义方式
- **`#define`**：预处理指令，无类型检查，全局替换
- **`const`**：编译期常量，有类型检查，作用域可控，推荐优先使用

#### 3. 整数类型体系（重点掌握）

| 类型 | 大小(字节) | 有符号范围 | 无符号范围 | 字面量后缀 |
|------|-----------|-----------|-----------|-----------|
| `char` | 1 | -128 ~ 127 | 0 ~ 255 | - |
| `short` | 2 | -32768 ~ 32767 | 0 ~ 65535 | - |
| `int` | 4 | -2^31 ~ 2^31-1 | 0 ~ 2^32-1 | - |
| `long` | 4(Win) / 8(Linux64) | 平台相关 | 平台相关 | `l`/`L` |
| `long long` | 8 | -2^63 ~ 2^63-1 | 0 ~ 2^64-1 | `ll`/`LL` |

**补码截断机制**（核心难点）：
- 有符号：对 `2^n` 取模后，若结果 > `2^(n-1)-1`，再减去 `2^n`
- 无符号：直接对 `2^n` 取模
- 示例：`signed short num = 32768` → 截断后为 `-32768`

#### 4. 浮点类型
- **float**：4字节，6-7位有效数字，字面量后缀 `f`/`F`
- **double**：8字节，15-17位有效数字，默认浮点字面量类型
- **注意**：`float a = 3.14` 会触发隐式转换，建议写 `float a = 3.14f`

#### 1. 固定宽度整数类型（int_t 和 uint_t）

```cpp
#include <cstdint>

int32_t num1 = 1;        // 4字节有符号整数
uint32_t num2 = 1u;      // 4字节无符号整数
int64_t num3 = 1LL;      // 8字节有符号整数
uint64_t num4 = 1ULL;    // 8字节无符号整数
```
- **关键理解**：固定宽度类型大小跨平台一致，解决了 `long` 类型平台依赖性问题

#### 2. 转义字符

| 转义序列 | 含义 | ASCII码 |

| `\xhh` | 十六进制转义 | 可变 |
| `\uXXXX` | Unicode转义 | 可变 |

#### 3. C语言风格 vs C++语言风格字符串

| 特性 | C语言风格 | C++语言风格 |
|------|----------|------------|
| 类型 | `char[]` | `std::string` |
| 内存管理 | 手动管理 | 自动管理 |
| 安全性 | 低（缓冲区溢出） | 高（自动检查） |
| 拼接 | `strcat()` | `+` 运算符 |
| 比较 | `strcmp()` | `==`, `!=`, `<`, `>` |
| 查找 | `strchr()` | `find()` |

#### 4. 布尔类型与逻辑运算符

```cpp
bool flag = true;
std::cout << std::boolalpha << flag << std::endl;  // 输出 true

// 逻辑运算符
!flag        // 逻辑非
flag1 && flag2  // 逻辑与（短路求值）
flag1 || flag2  // 逻辑或（短路求值）
```
- **短路求值**：`&&` 第一个操作数为 `false` 时，第二个操作数不执行；`||` 第一个操作数为 `true` 时，第二个操作数不执行
- **隐式转换**：非零整数 → `true`，零 → `false`；`true` → `1`，`false` → `0`

### 核心知识点回顾

#### 1. 输入流 cin

```cpp
// 基本输入
int num;
std::cin >> num;

// getline 读取整行
std::string line;
std::getline(std::cin, line);

// 连续读取多个值
int a, b, c;
std::cin >> a >> b >> c;
```

**cin 成员函数**：

| 成员函数 | 功能 |
|---------|------|
| `cin.get()` | 读取单个字符（包括空白字符） |
| `cin.getline()` | 读取一行到 char 数组 |
| `cin.ignore()` | 忽略指定数量的字符 |
| `cin.peek()` | 查看下一个字符但不提取 |
| `cin.putback()` | 将字符放回输入流 |
| `cin.read()` | 读取指定数量的字符 |
| `cin.gcount()` | 返回最后一次读取的字符数 |
| `cin.clear()` | 清除错误标志 |

#### 2. 缓冲区残留问题（重点）

```cpp
int age;
std::string name;

std::cin >> age;
// 此时缓冲区中残留 '\n'，直接调用 getline 会读到空字符串！
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');  // 解决方法
std::getline(std::cin, name);
```

- **问题**：`cin >>` 读取后换行符 `'\n'` 留在缓冲区，导致后续 `getline()` 读取到空字符串
- **解决方案**：使用 `cin.ignore()` 清理缓冲区

#### 3. 输出流 cout

```cpp
// 格式化输出
std::cout << std::boolalpha << true << std::endl;        // 输出 true
std::cout << std::hex << 255 << std::endl;                // 输出 ff
std::cout << std::fixed << std::setprecision(2) << 3.14159 << std::endl;  // 输出 3.14
std::cout << std::setw(10) << std::setfill('*') << "Hi" << std::endl;     // 输出 ********Hi
```

**`endl` vs `'\n'`**：
- `endl`：输出换行符并刷新缓冲区（性能开销较大）
- `'\n'`：只输出换行符（性能更好，推荐使用）

**常用 I/O 操纵符**：

| 操纵符 | 功能 |
|--------|------|
| `std::boolalpha` | 布尔值输出为 true/false |
| `std::dec/hex/oct` | 十进制/十六进制/八进制输出 |
| `std::fixed` | 固定小数位数 |
| `std::scientific` | 科学计数法 |
| `std::setprecision(n)` | 设置精度 |
| `std::setw(n)` | 设置输出宽度 |
| `std::setfill(c)` | 设置填充字符 |
| `std::left/right/internal` | 对齐方式 |

#### 4. 算术运算符

```cpp
int a = 10, b = 3;
a + b;   // 13
a - b;   // 7
a * b;   // 30
a / b;   // 3（整数除法，截断小数）
a % b;   // 1（取模，仅用于整数）

// 整数除法转浮点数的方法
static_cast<double>(a) / b;  // 3.333...（推荐）
a / (double)b;               // 3.333...
a * 1.0 / b;                 // 3.333...
```

- **整数除法截断规则**：向零取整（`-7 / 2 = -3`，不是 `-4`）
- **负数取模**：余数符号与被除数相同（`-10 % 3 = -1`）

#### 5. 递增递减运算符

```cpp
int x = 5;
++x;   // 前置：先加1，再返回新值（x=6，表达式结果=6）
x++;   // 后置：先返回原值，再加1（x=7，表达式结果=6）
```

- **前置 `++var`**：先修改，再返回引用（性能更好）
- **后置 `var++`**：先保存副本，再修改，返回副本（有额外开销）

#### 6. 赋值运算符

```cpp
int a = 10;
a += 5;   // a = a + 5 = 15
a -= 3;   // a = a - 3 = 12
a *= 2;   // a = a * 2 = 24
a /= 4;   // a = a / 4 = 6
a %= 4;   // a = a % 4 = 2

// 链式赋值（从右到左）
int x, y, z;
x = y = z = 5;   // 先z=5，再y=z，再x=y
```

- **复合赋值更高效**：左值只求值一次（如 `arr[i] += 5` 比 `arr[i] = arr[i] + 5` 高效）

#### 7. 比较运算符

```cpp
5 == 5;   // true
5 != 3;   // true
5 > 3;    // true
5 < 3;    // false
5 >= 5;   // true
5 <= 3;   // false
```

- **连续比较陷阱**：`3 < num < 10` 在 C++ 中是 `(3 < num) < 10`，不是数学中的范围判断
- **正确写法**：`3 < num && num < 10`
- **浮点数比较**：避免直接用 `==`，应使用 `abs(a - b) < epsilon`

#### 8. 逻辑运算符

```cpp
!true;         // false
true && false; // false（逻辑与，短路求值）
true || false; // true（逻辑或，短路求值）
```

- **短路求值**：
  - `&&`：第一个为 `false` 时，第二个不执行
  - `||`：第一个为 `true` 时，第二个不执行
- **优先级**：`!` > `&&` > `||`

#### 9. 位运算符

```cpp
0b1010 & 0b1100;  // 0b1000（按位与：清零、提取位）
0b1010 | 0b0100;  // 0b1110（按位或：设置位）
0b1010 ^ 0b1100;  // 0b0110（按位异或：翻转位、交换变量）
~0b1010;          // 按位非：取反
0b0010 << 2;      // 0b1000（左移：快速乘法）
0b1000 >> 2;      // 0b0010（右移：快速除法）
```

- **逻辑右移 vs 算术右移**：无符号整数补0（逻辑），有符号整数补符号位（算术）
- **位运算 vs 逻辑运算**：位运算返回整数，不支持短路求值；逻辑运算返回 bool，支持短路求值

### 核心知识点回顾

#### 1. 选择结构

```cpp
// if-else if-else 多分支
if (score >= 90) {
    // A
} else if (score >= 80) {
    // B
} else {
    // F
}

// 三目运算符
int max = (a > b) ? a : b;

// switch语句（离散值判断）
switch (day) {
    case 1: std::cout << "Monday"; break;
    case 2: std::cout << "Tuesday"; break;
    default: std::cout << "Invalid"; break;
}
```

- **三目运算符右结合性**：`a ? b : c ? d : e` 等价于 `a ? b : (c ? d : e)`
- **switch case穿透**：缺少 `break` 时会继续执行下一个case
- **switch限制**：表达式只能是整型或枚举，不能是浮点数或字符串

#### 2. 循环结构

```cpp
// while：先判断后执行
while (condition) { ... }

// do-while：先执行后判断（至少执行一次）
do { ... } while (condition);

// for：三部分合一
for (init; condition; update) { ... }

// range-for（C++11）：遍历容器
for (int val : arr) { ... }
for (int& val : arr) { val *= 2; }  // 引用修改
```

**三种循环对比**：

| 循环类型 | 特点 | 适用场景 |
|---------|------|---------|
| `while` | 先判断后执行 | 不确定循环次数 |
| `do-while` | 先执行后判断 | 至少执行一次 |
| `for` | 三部分合一 | 确定循环次数 |
| `range-for` | 简洁遍历 | 遍历容器/数组 |

#### 3. 跳转语句

```cpp
// break：跳出当前循环或switch
for (int i = 0; i < 10; ++i) {
    if (i == 5) break;  // 跳出for循环
}

// continue：跳过本次迭代
for (int i = 0; i < 10; ++i) {
    if (i % 2 == 0) continue;  // 跳过偶数
    std::cout << i;  // 只输出奇数
}

// goto：无条件跳转（不推荐）
goto label;
// ...
label: statement;
```

**跳转语句对比**：

| 语句 | 作用 | 适用范围 | 注意事项 |
|------|------|---------|---------|
| `break` | 跳出当前循环/switch | 循环、switch | 只跳一层 |
| `continue` | 跳过本次迭代 | 循环 | 不终止循环 |
| `goto` | 无条件跳转 | 任意位置 | 不推荐使用 |
| `return` | 退出函数 | 函数 | 可带返回值 |

### 学习建议

#### 代码风格优化
1. **优先使用for循环**：已知循环次数时，for循环最清晰
2. **避免使用goto**：使用标志变量或提取函数代替goto跳出多层循环
3. **使用range-for遍历容器**：C++11的range-for比传统for更简洁安全
4. **switch中始终使用default**：处理未预期的情况，增强健壮性
5. **注意continue在while中的陷阱**：确保循环变量更新不被continue跳过

### 核心知识点回顾

#### 1. 数组的定义

```cpp
// 基本定义
int arr[5];          // 定义包含5个int的数组（未初始化，局部变量为垃圾值）

// 不同类型的数组
int8_t arr1[10];     // 字符数组（10字节）
double arr2[3];      // 浮点数组（24字节）
bool arr3[4];        // 布尔数组（4字节）

// 使用const表达式作为数组大小
const int SIZE = 5;
int arr4[SIZE];

// C++03：数组大小必须是常量表达式
// int n = 5;
// int arr5[n];  // 错误！n不是常量
```

#### 2. 数组的初始化

| 方式 | 示例 | 说明 |
|------|------|------|
| 完全初始化 | `int arr[5] = {1, 2, 3, 4, 5};` | 所有元素都有值 |
| 部分初始化 | `int arr[5] = {1, 2, 3};` | 未初始化元素默认为0 |
| 省略大小 | `int arr[] = {1, 2, 3, 4};` | 编译器自动推断大小为4 |
| 全部为0 | `int arr[5] = {0};` | 所有元素初始化为0 |
| C++11花括号 | `int arr[5]{1, 2, 3, 4, 5};` | 不允许窄化转换 |
| 字符数组 | `char str[] = "Hello";` | 自动添加'\0'，大小为6 |

#### 3. 数组元素的访问

```cpp
int arr[5] = {10, 20, 30, 40, 50};

// 通过下标访问（下标从0开始）
arr[0];   // 10（第一个元素）
arr[4];   // 50（最后一个元素）

// 修改元素
arr[1] = 200;

// 循环遍历
for (int i = 0; i < 5; ++i) {
    std::cout << arr[i] << " ";
}

// range-for遍历（C++11）
for (int val : arr) {
    std::cout << val << " ";
}

// 指针遍历
for (int* ptr = arr; ptr < arr + 5; ++ptr) {
    std::cout << *ptr << " ";
}
```

**数组与指针的关系**：
- `arr` 等价于 `&arr[0]`（数组名就是首元素地址）
- `arr[i]` 等价于 `*(arr + i)`（指针算术）
- `*arr` 等价于 `arr[0]`（解引用）

#### 4. 数组大小的计算

```cpp
int arr[5] = {1, 2, 3, 4, 5};

// 总大小（字节）
sizeof(arr);          // 20（5 * 4）

// 单个元素大小（字节）
sizeof(arr[0]);       // 4

// 元素个数（最常用）
sizeof(arr) / sizeof(arr[0]);  // 5

// 封装为宏
#define ARRAY_SIZE(arr) (sizeof(arr) / sizeof(arr[0]))
ARRAY_SIZE(arr);      // 5
```

**陷阱**：数组作为函数参数时退化为指针！
```cpp
void printArray(int arr[]) {
    // arr在这里是指针，不是数组！
    sizeof(arr);  // 返回指针大小（8字节），不是数组大小！
}
```
**解决方案**：始终将数组大小作为单独的参数传递。

#### 5. 数组的常见问题和注意事项

| 问题 | 说明 |
|------|------|
| 下标越界 | C++不检查数组边界，越界访问是未定义行为 |
| 数组大小必须是常量 | C++03要求，C++11+支持VLA但不推荐 |
| 局部数组未初始化 | 全局数组默认初始化为0，局部数组包含垃圾值 |
| 数组退化指针 | 作为函数参数时退化为指针，sizeof返回指针大小 |
| C风格字符串 | 必须以'\0'结尾，否则会越界访问 |

#### 代码风格优化
1. **始终初始化数组**：避免未初始化的局部数组，使用 `{0}` 或完整初始化
2. **使用ARRAY_SIZE宏**：方便计算数组元素个数，避免硬编码
3. **优先使用range-for**：遍历数组时，range-for比传统for更简洁安全
4. **传递数组时带上大小**：函数参数中同时传递数组和大小，避免退化指针问题
5. **使用const修饰只读数组**：如果数组内容不修改，使用 `const int arr[]`
